cell.

Rejillas multiescalares asimétricas.

cell es un prototipo de plataforma para la gestión de rejillas de datos geográficos multiescalares y asimétricas. Es el resultado de mi Tesis Doctoral en Geografía, realizada en el Universidad de Sevilla, Departamento de Geografía Física y AGR, entre los años 2016-2021, y titulada "Plataforma cloud para la integración espacial de geoinformación en estructuras de teselas multiescalares asimétricas, su análisis y su visualización", financiada por una beca de investigación del Ministerio de Ciencia e Innovación del Gobierno de España BES-2015-072172 en el proyecto CSO2014-51994-P.

Este prototipo se ha seguido desarrollando, incorporando nuevas variables y adaptándolo a nuevos objetivos temáticos, en el contexto de los siguientes proyectos de investigación:

  • Plataforma Cloud para la Integración Espacial de Geoinformación ambiental y socioeconómica (Spatial Big Data) y Clientes Web Interactivos (Geovisores, DashBoard) para su difusión, acceso, explotación y análisis (Maching Learning). Proyecto singular de actuaciones de transferencia CEI RIS3. Junta de Andalucía. Andalucía Tech CEI-9.
  • Prototipo Cloud de Evaluación de Riesgos a Través de la Integración Espacial de Geodatos (Grid Multiescalares) y Clientes Web (Dashboard) para su Gestión y Medidas de Adaptación. Proyecto Plan Estatal I+D+i Retos. Ministerio de Ciencia e Innovación. PID2019-106834RB-I00.
  • E-Infraestructura (Cloud Computing) para la Integración Espacial de Variables Antrópicas en el Cálculo de la Vulnerabilidad ante Riesgos Costeros y HerramientasWeb (Dashboard) para su Gestión en Andalucía (Clodrisk_A). Proyecto I+D+i Frontera. Junta de Andalucía. P18-FR-2574.
  • Scientific Infrastructures for Global Change Monitoring and Adaptation in Andalusia (INDALO). Convocatoria de Operaciones cofinanciadas FEDER (Programa Operativo Plurirregional de España) para actividades relacionadas con LifeWatch ERIC. LIFEWATCH-2019-04-AMA-01.

Esta página es un compendio de los recursos generados durante los trabajos para la realización de la Tesis Doctoral, que esperemos se vayan ampliando con el tiempo para poner a disposición de los desarrolladores TIG todos los recursos necesarios para utilizar la plataforma por sus propios medios.

Básicamente, esta página recoge los enlaces a tres tipos de recursos:

  • visores: visores desarrollados como muestra de la plataforma;
  • repositorios de datos: repositorio de tratamiento de datos utilizados para ejemplificar las posibilidades de la plataforma;
  • repositorios de código: repositorios de código fuente de los distintos componentes de la plataforma.

Visores

A continuación se detallan los visores creados para demostrar las capacidades de la plataforma:

  • visor con widgets de población: realizado con la plataforma CARTO, este visor permite la exploración de la información de rejilla mediante histogramas (enlace);
  • visor con widgets de resultados de Machine Learning: realizado también CARTO, este visor permite explorar los resultados de los tests hechos con los procedimientos de K-Means y Random Forest sobre los datos de población (enlace);
  • visor con widgets de resultados de Machine Learning, versión con ortofoto: versión con ortofoto de fondo del anterior visor, dado que la tipología urbana clasificada por el Random Forest se aprecia mejor con dicha base cartográfica. Por lo demás, idéntico al anterior (enlace).

Repositorios de datos

Estos repositorios de datos contienen el código de manipulación ETL de los datos incorporados en la plataforma a modo de prueba. Todos ellos se pueden encontrar en los siguientes enlaces:

  • malkab-phd-data-cell_raw_data: constructor principal de la base de datos de datos originales, donde se realiza todo el proceso ETL y de lectura y proceso de datos de teselación (enlace);
  • malkab-phd-data-proyecto_nacional_data: proceso de datos de catastro (enlace);
  • malkab-phd-data-context_data: proceso de datos de contexto (provincias, municipios, etc.) (enlace);
  • malkab-phd-data-hic: proceso de datos de Hábitats de Interés Comunitario (enlace);
  • malkab-phd-data-población: proceso de datos de población (enlace).

Repositorios de código

Todos estos repositorios contienen el código de las distintas librerías y componentes de la plataforma:

  • cell: repositorio principal de la plataforma, con todos los componentes de microservicios (enlace);
  • appian: framework sobre Express para construir API de servicios web con más comodidad y estandarización (enlace);
  • rewhitt: framework sobre Docker, PostgreSQL, Redis y TypeScript para controlar procesos distribuidos (enlace);
  • rxpg: librería cliente PostgreSQL para TypeScript basado en RxJs (enlace);
  • rxredis: librería cliente Redis para TypeScript basado en RxJs (enlace);
  • node-logger: librería de logging para TypeScript (enlace);
  • node-utils: conjunto de utilidades de carácter general para la plataforma Node (enlace);
  • ts-utils: conjunto de utilidades de carácter general para el lenguaje TypeScript (enlace).
